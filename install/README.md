# نايل سنتر - نظام طلبات العملاء
## Nile Center - Customer Requests System

### المتطلبات
- Windows 10/11
- XAMPP (Apache + MySQL + PHP)
- متصفح حديث (Chrome, Firefox, Edge)

---

### خطوات التنصيب

#### 1. تنصيب XAMPP
1. حمل XAMPP من: https://www.apachefriends.org
2. اعمل Install في المسار: C:\xampp
3. مهم: لو ظهر تحذير UAC، اضغط OK واستمر
4. شغل Apache و MySQL من XAMPP Control Panel
5. لو Apache مشتغلش، غير الـ Port لـ 8080:
   - Config -> httpd.conf -> غير Listen 80 لـ Listen 8080
   - غير ServerName localhost:80 لـ ServerName localhost:8080

#### 2. إنشاء قاعدة البيانات
1. افتح المتصفح على: http://localhost/phpmyadmin (أو http://localhost:8080/phpmyadmin)
2. اضغط على New من الشمال
3. سميها: nile_center
4. اختار utf8mb4_general_ci من القائمة
5. اضغط Create
6. اختار قاعدة البيانات nile_center
7. اضغط على تبويب Import
8. اختار ملف database/schema.sql
9. اضغط Go

#### 3. نسخ ملفات النظام
1. افتح مجلد: C:\xampp\htdocs
2. اعمل مجلد جديد: nile-center-system
3. انسخ كل الملفات من المشروع للمجلد ده

#### 4. تعديل الإعدادات
1. افتح ملف: core/config.php
2. عدل السطر ده:
   define('APP_URL', 'http://26.201.9.238/nile-center-system');
3. لو غيرت Port Apache لـ 8080:
   define('APP_URL', 'http://26.201.9.238:8080/nile-center-system');

#### 5. الدخول للنظام
1. افتح المتصفح على: http://26.201.9.238/nile-center-system
   (أو مع Port 8080 لو غيرته)
2. يوزر: admin
3. پاسورد: admin123
4. غير الباسورد فوراً من إدارة المستخدمين!
